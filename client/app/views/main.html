Tagimacate all the things!

Text input item tagging.

Here's why you should use the tagimacator:

  • Optimized: Tagimacator doesn't block browser execution when rendering or querying large data sets!
  • Modular: Tagimcator utilizes angular directives for awesome portability.
  • Configurable: Have multiple instances of tagimacators with various configurations!

Check out the demos page for instructions on installing and using!

Installation

Install via Bower:

  
bower install tagimacator

Once installed, source tagimacator in your index.html and include the tagimacator in your app's module declaration: 

  
angular.module('myApp', ['tagimacator']);

Tagimacator uses Sass, so you'll have to:

  
@import 'tagimacator';

in your main scss file.

After that, you are free to use the tagimacator directive anywhere!

NOTE:

You may have to change the templateUrl property in the tagimacator directive to match your app's file system.
